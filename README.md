# 🚀 Premium Digital Tools

## 📌 Project Overview
Premium Digital Tools is a modern web application where users can explore, purchase, and manage digital products such as AI tools, templates, and productivity resources. The platform provides a smooth user experience with dynamic product listing, cart functionality, and responsive design.

---

## 📝 Description
This project allows users to browse premium digital products, view details, and add items to a cart. Users can manage their cart by adding or removing products and see the total cost instantly. The UI is clean, responsive, and designed for a seamless experience across all devices.

## ✨ Key Features
1. 🛍️ **Dynamic Product Listing**  
   Products are loaded dynamically from a JSON file and displayed in responsive cards.

2. 🛒 **Cart Management System**  
   Users can add/remove items from the cart with real-time updates and total price calculation.

3. 🎯 **Responsive & Modern UI**  
   Fully responsive design with gradient UI, interactive buttons, and smooth user experience.
